# Tetris

普通のテトリスです。

## 遊べる

遊べます。→[URL](https://nekotonn.github.io/Tetris/Build)  
キーボードが必要です。

### 操作

- ←キー: 左に移動
- →キー: 右に移動
- ↓キー: ソフトドロップ
- ↑キー: ハードドロップ
- Zキー: 左回転
- Xキー: 右回転
- SPACEキー: ホールド
- Enterキー: GAMEOVERからの復帰
